What is HTML?

HTML is a markup language used to create web pages. It stands for HyperText Markup Language. HTML uses tags to mark up text, images, and other elements on a web page.

What is Semantic HTML?

Semantic HTML is a way of using HTML to mark up the meaning of the content on a web page. This can help screen readers and other assistive technologies to understand the content of the page and make it easier for users to navigate.

Commonly used semantic tags

Here are some of the most commonly used semantic tags:

  • <header>: The header tag is used to mark up the header of a web page.
  • <nav>: The nav tag is used to mark up a navigation section of a web page.
  • <article>: The article tag is used to mark up an independent piece of content, such as a blog post, news article, or product review.
  • <aside>: The aside tag is used to mark up content that is related to the main content of the page, but is not essential.
  • <footer>: The footer tag is used to mark up the footer of a web page.
